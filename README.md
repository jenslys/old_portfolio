## Run Locally

Clone the project

```bash
  git clone https://github.com/jenslys/lystad.io.git
```

Go to the project directory

```bash
  cd lystad.io
```

Install dependencies

```bash
  yarn
```

Start the local server

```bash
  yarn dev
```
